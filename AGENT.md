# Agent Guide

## Project
- Python project managed with uv.
- Entry point: `main.py`.

## Environment
- Create and sync env: `uv sync`
- Run: `uv run python main.py`

## Dependencies
- Add a dependency: `uv add <package>`
- Update lockfile after edits: `uv sync`

## Notes
- Keep changes minimal and focused (KISS/YAGNI).
